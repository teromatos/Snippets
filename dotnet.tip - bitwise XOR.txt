Binary Math

0 + 0 = 0
0 + 1 = 1
1 + 1 = 10

Binary to Decimal Conversion

00010011 = 0*2**7 + 0*2**6 + 0*2**5 + 1*2**4 +
           0*2**3 + 0*2**2 + 1*2**1 + 1*2**0
         = 0 + 0 + 0 + 16 + 0 + 0 + 2 + 1
         = 19


Hexadecimal to Decimal Conversion

3F = 3*16**1 + 15*16**0
   = 63 (01111111 in binary)
